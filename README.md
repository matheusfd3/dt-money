## Primeiro Acesso

Clone o projeto

```bash
  git clone https://github.com/matheusfd3/dt-money.git
```

Entre no diretório do projeto

```bash
  cd dt-money
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev:server
```
E também

```bash
  npm run dev
```
